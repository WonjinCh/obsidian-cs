# 플로이드 워셜

상위개념: [[00_동적 프로그래밍]] | 인덱스: [[알고리즘 개요]]

---

## 개요

**모든 정점 쌍** 사이의 최단 경로를 한 번에 구하는 DP 기반 최단경로 알고리즘. 음의 가중치도 허용 (음의 사이클 없을 때).

---

## 핵심 아이디어

- `dist[i][j]`: i에서 j로 가는 최단 거리
- 중간 경유 정점 k를 0~V-1까지 순서대로 고려
- 점화식: `dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])`

---

## 자료구조

- [[그래프]] — 인접 행렬
- 2차원 DP 배열

---

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 시간 | O(V³) |
| 공간 | O(V²) |

> V가 수백 이상이면 시간 초과 주의. 자세한 내용: [[시간복잡도와 공간복잡도]]

---

## 동작 원리 (단계별)

```
초기화:
  dist[i][i] = 0
  dist[i][j] = 간선 가중치 (없으면 INF)

k=0 (0번 정점 경유):
  모든 (i,j)에 대해
  dist[i][j] = min(dist[i][j], dist[i][0] + dist[0][j])

k=1 (1번 정점 경유):
  모든 (i,j)에 대해
  dist[i][j] = min(dist[i][j], dist[i][1] + dist[1][j])
...반복...
```

## 예시 코드

### Java
```java
final int INF = Integer.MAX_VALUE / 2;
int V = 4;
int[][] dist = new int[V][V];

for (int[] row : dist) Arrays.fill(row, INF);
for (int i = 0; i < V; i++) dist[i][i] = 0;

// 간선 입력
dist[0][1] = 5; dist[0][3] = 10;
dist[1][2] = 3; dist[2][3] = 1;

// 플로이드 워셜
for (int k = 0; k < V; k++)
    for (int i = 0; i < V; i++)
        for (int j = 0; j < V; j++)
            if (dist[i][k] + dist[k][j] < dist[i][j])
                dist[i][j] = dist[i][k] + dist[k][j];
```

### JavaScript
```javascript
function floydWarshall(n, edges) {
    const INF = Infinity;
    const dist = Array.from({length: n}, (_, i) =>
        Array.from({length: n}, (_, j) => i === j ? 0 : INF)
    );

    for (const [u, v, w] of edges) dist[u][v] = w;

    for (let k = 0; k < n; k++)
        for (let i = 0; i < n; i++)
            for (let j = 0; j < n; j++)
                if (dist[i][k] + dist[k][j] < dist[i][j])
                    dist[i][j] = dist[i][k] + dist[k][j];

    return dist;
}
```

---

## 언제 사용하나

- **모든 쌍** 최단경로가 필요할 때
- V가 작을 때 (V ≤ 500 정도)
- 음의 가중치가 있을 때 (음의 사이클은 없어야 함)
- 음의 사이클 감지: `dist[i][i] < 0`이면 존재

---

## 주의사항

- INF 설정 시 덧셈 오버플로 주의 → `INF = Integer.MAX_VALUE / 2`
- O(V³)이라 V ≥ 1000이면 시간 초과 가능
- 단일 출발점 최단경로는 [[03_다익스트라]] 사용
