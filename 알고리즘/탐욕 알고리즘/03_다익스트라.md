# 다익스트라

상위개념: [[00_탐욕 알고리즘]] | 인덱스: [[알고리즘 개요]]

---

## 개요

**단일 출발점**에서 다른 모든 정점까지의 최단 경로를 구하는 알고리즘. **양의 가중치** 그래프에서만 동작.

---

## 핵심 아이디어

- 시작 정점의 거리를 0, 나머지는 INF로 초기화
- **우선순위 큐(Min-Heap)** 로 최소 거리 정점을 빠르게 선택
- 선택된 정점의 인접 정점 거리 갱신 (Relaxation)
- 이미 처리된 정점은 스킵

---

## 자료구조

- [[그래프]] — 인접 리스트 (가중치 그래프)
- 우선순위 큐 (Min-Heap)
- 거리 배열 `dist[]`

---

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 시간 | O(E log V) — 우선순위 큐 |
| 공간 | O(V + E) |

## 동작 원리 (단계별)

```
시작: 0, 그래프: 0→1(4), 0→2(1), 2→1(2), 1→3(1)

초기: dist=[0, INF, INF, INF], PQ=[(0,0)]

Step1: pop(0,0) → dist[0]=0 확정
       인접: dist[1]=4, dist[2]=1
       PQ=[(1,2),(4,1)]

Step2: pop(1,2) → dist[2]=1 확정
       인접: dist[1]=min(4, 1+2)=3
       PQ=[(3,1),(4,1)]

Step3: pop(3,1) → dist[1]=3 확정
       인접: dist[3]=min(INF, 3+1)=4

Step4: pop(4,1) → dist[1]=3 이미 처리 → 스킵

결과: dist=[0, 3, 1, 4]
```

## 예시 코드

### Java
```java
int[] dijkstra(int V, List<int[]>[] graph, int start) {
    int[] dist = new int[V];
    Arrays.fill(dist, Integer.MAX_VALUE);
    dist[start] = 0;

    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
    pq.offer(new int[]{0, start});

    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int d = curr[0], node = curr[1];
        if (d > dist[node]) continue;  // lazy deletion

        for (int[] next : graph[node]) {
            int newDist = dist[node] + next[1];
            if (newDist < dist[next[0]]) {
                dist[next[0]] = newDist;
                pq.offer(new int[]{newDist, next[0]});
            }
        }
    }
    return dist;
}
```

### JavaScript
```javascript
function dijkstra(V, graph, start) {
    const dist = new Array(V).fill(Infinity);
    dist[start] = 0;
    const pq = [[0, start]];

    while (pq.length > 0) {
        pq.sort((a, b) => a[0] - b[0]);
        const [d, node] = pq.shift();
        if (d > dist[node]) continue;

        for (const [next, w] of graph[node]) {
            const newDist = dist[node] + w;
            if (newDist < dist[next]) {
                dist[next] = newDist;
                pq.push([newDist, next]);
            }
        }
    }
    return dist;
}
```

## 언제 사용하나

- **단일 출발점** 최단 경로 (양의 가중치)
- 네트워크 라우팅, 지도 최단경로
- BFS로 해결 안 되는 가중 그래프

---

## 주의사항

- **음의 가중치 불가** → 벨만-포드 알고리즘 사용
- `if (d > dist[node]) continue` 처리 필수 (lazy deletion)
- 모든 쌍 최단경로 → [[01_플로이드 워셜]]
