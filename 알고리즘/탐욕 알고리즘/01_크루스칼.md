# 크루스칼

상위개념: [[00_탐욕 알고리즘]] | 인덱스: [[알고리즘 개요]]

---

## 개요

**간선(Edge)** 을 가중치 오름차순으로 정렬하여, 사이클을 형성하지 않는 간선을 차례로 선택해 **최소 신장 트리(MST)** 를 구하는 탐욕 알고리즘.

---

## 핵심 아이디어

- **Union-Find(Disjoint Set)** 으로 사이클 감지
- 간선 오름차순 정렬 후 순서대로 검토
- 두 정점이 같은 집합이면 → 사이클 → 스킵
- 다른 집합이면 → MST에 추가 → Union

---

## 자료구조

- [[그래프]] — 엣지 리스트
- Union-Find (경로 압축 + 랭크)

---

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 시간 | O(E log E) — 정렬 지배 |
| 공간 | O(V + E) |

> 자세한 내용: [[시간복잡도와 공간복잡도]]

## 동작 원리 (단계별)

```
정점: 4개 (0,1,2,3)
간선 (정렬 후): [(0,1,1),(1,2,2),(0,2,3),(1,3,4),(2,3,5)]

Step1: (0,1,1) → 다른 집합 → MST 추가, Union(0,1)
Step2: (1,2,2) → 다른 집합 → MST 추가, Union(1,2)
Step3: (0,2,3) → 같은 집합(0,1,2) → 스킵
Step4: (1,3,4) → 다른 집합 → MST 추가, Union(1,3)

MST 비용 = 1+2+4 = 7  (V-1 = 3개 간선)
```

## 예시 코드

### Java
```java
int[] parent, rank;

int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
}

boolean union(int a, int b) {
    int ra = find(a), rb = find(b);
    if (ra == rb) return false;
    if (rank[ra] < rank[rb]) { int t = ra; ra = rb; rb = t; }
    parent[rb] = ra;
    if (rank[ra] == rank[rb]) rank[ra]++;
    return true;
}

int kruskal(int V, int[][] edges) {
    parent = new int[V]; rank = new int[V];
    for (int i = 0; i < V; i++) parent[i] = i;
    Arrays.sort(edges, (a, b) -> a[2] - b[2]);

    int total = 0, count = 0;
    for (int[] edge : edges) {
        if (union(edge[0], edge[1])) {
            total += edge[2];
            if (++count == V - 1) break;
        }
    }
    return total;
}
```

### JavaScript
```javascript
function kruskal(V, edges) {
    const parent = Array.from({length: V}, (_, i) => i);
    const rank = new Array(V).fill(0);

    function find(x) {
        if (parent[x] !== x) parent[x] = find(parent[x]);
        return parent[x];
    }

    function union(a, b) {
        const ra = find(a), rb = find(b);
        if (ra === rb) return false;
        if (rank[ra] >= rank[rb]) parent[rb] = ra;
        else parent[ra] = rb;
        if (rank[ra] === rank[rb]) rank[ra]++;
        return true;
    }

    edges.sort((a, b) => a[2] - b[2]);
    let total = 0;
    for (const [u, v, w] of edges) {
        if (union(u, v)) total += w;
    }
    return total;
}
```

## 언제 사용하나

- **MST** 구하기
- 간선 수가 적은 희소 그래프 (Sparse Graph)

---

## 주의사항

- V-1개 간선을 선택하면 MST 완성
- 연결 그래프가 아니면 MST가 존재하지 않음
- Union-Find 없이 구현하면 O(V²) → 필수 사용
