## 개요

**정점(Vertex)** 기준으로 현재 MST에 포함된 정점들과 연결된 간선 중 최소 가중치 간선을 반복 선택하여 **최소 신장 트리(MST)** 를 구하는 탐욕 알고리즘.

---

## 핵심 아이디어

- 임의의 시작 정점에서 출발
- **우선순위 큐(Min-Heap)** 로 최소 비용 간선 빠르게 선택
- 이미 방문한 정점은 스킵
- V-1개 간선 선택 시 완료

---

## 자료구조

- [[그래프]] — 인접 리스트
- 우선순위 큐 (Min-Heap)
- 방문 배열

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 시간 | O(E log V) — 우선순위 큐 |
| 공간 | O(V + E) |

## 동작 원리 (단계별)

```
정점: 4개, 시작: 0
인접: 0-1(1), 0-2(3), 1-2(2), 1-3(4), 2-3(5)

Init: PQ=[(1,1),(3,2)], visited={0}

Step1: pop(1,1) → 1 미방문 → MST 추가, visited={0,1}
       PQ에 1의 인접 추가: (2,2),(4,3)

Step2: pop(2,2) → 2 미방문 → MST 추가, visited={0,1,2}

Step3: pop(3,2) → 2 방문됨 → 스킵

Step4: pop(4,3) → 3 미방문 → MST 추가

MST 비용 = 1+2+4 = 7
```

## 예시 코드

### Java
```java
int prim(int V, List<int[]>[] graph) {
    boolean[] visited = new boolean[V];
    PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
    pq.offer(new int[]{0, 0});  // {비용, 정점}

    int total = 0;
    while (!pq.isEmpty()) {
        int[] curr = pq.poll();
        int cost = curr[0], node = curr[1];
        if (visited[node]) continue;
        visited[node] = true;
        total += cost;

        for (int[] next : graph[node]) {
            if (!visited[next[0]]) pq.offer(new int[]{next[1], next[0]});
        }
    }
    return total;
}
```

### JavaScript
```javascript
function prim(V, graph) {
    const visited = new Array(V).fill(false);
    const pq = [[0, 0]];  // [cost, node]
    let total = 0;

    while (pq.length > 0) {
        pq.sort((a, b) => a[0] - b[0]);
        const [cost, node] = pq.shift();
        if (visited[node]) continue;
        visited[node] = true;
        total += cost;

        for (const [next, w] of graph[node]) {
            if (!visited[next]) pq.push([w, next]);
        }
    }
    return total;
}
```

## 언제 사용하나

- **MST** 구하기
- 간선 수가 많은 밀집 그래프 (Dense Graph)

## 주의사항

- 연결 그래프가 아니면 MST 불완전
- JS에서 PriorityQueue 없으면 sort() → O(E log E) 증가
