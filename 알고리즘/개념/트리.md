# 트리

인덱스: [[알고리즘 개요]]

---

## 정의

**사이클이 없는 연결 그래프**. V개의 정점과 V-1개의 간선으로 구성. 계층적 구조를 표현.

```
        1          ← 루트(Root)
       / \
      2   3        ← 내부 노드
     / \   \
    4   5   6      ← 리프(Leaf)
```

---

## 주요 용어

| 용어 | 설명 |
|------|------|
| 루트 (Root) | 최상위 노드 |
| 리프 (Leaf) | 자식이 없는 노드 |
| 깊이 (Depth) | 루트에서 해당 노드까지 거리 |
| 높이 (Height) | 트리의 최대 깊이 |
| 서브트리 (Subtree) | 특정 노드를 루트로 하는 하위 트리 |

## 트리 종류

### 이진 트리 (Binary Tree)
- 각 노드가 최대 2개의 자식
- 완전 이진 트리: 마지막 레벨 제외 모두 차 있음

### 이진 탐색 트리 (BST)
- 왼쪽 < 루트 < 오른쪽
- 탐색/삽입/삭제: 평균 O(log n), 최악 O(n)
- 균형 BST (AVL, Red-Black): 최악도 O(log n) 보장

```java
class Node {
    int val;
    Node left, right;
    Node(int val) { this.val = val; }
}

Node search(Node root, int target) {
    if (root == null || root.val == target) return root;
    return target < root.val ? search(root.left, target)
                              : search(root.right, target);
}
```

### 힙 (Heap)
- 완전 이진 트리 + 부모-자식 간 크기 관계
- **최소 힙**: 부모 ≤ 자식 → 루트가 최솟값
- **최대 힙**: 부모 ≥ 자식 → 루트가 최댓값
- 삽입/삭제: O(log n), 최솟값/최댓값 조회: O(1)
- 사용: [[03_다익스트라\|다익스트라]], [[02_프림\|프림]], [[01_크루스칼\|크루스칼]]의 우선순위 큐

```java
// Java 우선순위 큐 (Min-Heap)
PriorityQueue<Integer> minHeap = new PriorityQueue<>();
minHeap.offer(3); minHeap.offer(1); minHeap.offer(2);
System.out.println(minHeap.poll()); // 1

// Max-Heap
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());
```

```javascript
// Min-Heap — 직접 구현 필요 (기본 제공 없음)
const heap = [3, 1, 2];
heap.sort((a, b) => a - b);
console.log(heap.shift()); // 1
```

## 트리 순회 (DFS 기반)

```
       1
      / \
     2   3
    / \
   4   5

전위 (Pre-order):  루트→좌→우 = 1,2,4,5,3
중위 (In-order):   좌→루트→우 = 4,2,5,1,3  ← BST 정렬 순서
후위 (Post-order): 좌→우→루트 = 4,5,2,3,1
레벨 (BFS):        1,2,3,4,5
```

```java
void inorder(Node root) {
    if (root == null) return;
    inorder(root.left);
    System.out.print(root.val + " ");
    inorder(root.right);
}
```
