# 시간복잡도와 공간복잡도

인덱스: [[알고리즘 개요]]

---

## 정의

- **시간복잡도**: 입력 크기 n에 따른 연산 횟수의 증가율
- **공간복잡도**: 알고리즘 실행에 필요한 메모리 사용량

---

## Big-O 표기법

최악의 경우(Worst Case)를 기준으로 상한(Upper Bound)을 표현.

```
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2^n) < O(n!)
```

## 시간제한 역산 가이드

> 1초에 약 10^8 ~ 10^9 연산 처리 가능

| 데이터 크기 n | 허용 복잡도 | 알고리즘 후보 |
|------------|-----------|------------|
| n ≤ 10 | O(n!) | 순열 완전탐색 |
| n ≤ 25 | O(2^n) | 부분집합, 백트래킹 |
| n ≤ 100 | O(n³) | 플로이드 워셜, 3중 루프 |
| n ≤ 1,000 | O(n²) | 기본 정렬, DP |
| n ≤ 100,000 | O(n log n) | 병합 정렬, 이분 탐색, BFS/DFS |
| n ≤ 1,000,000 | O(n) | 선형 탐색, 그리디 |
