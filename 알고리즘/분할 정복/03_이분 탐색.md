# 이분 탐색

상위개념: [[00_분할 정복]] | 인덱스: [[알고리즘 개요]]

---

## 개요

**정렬된 배열**에서 탐색 범위를 절반씩 줄여가며 목표값을 찾는 탐색 알고리즘.

---

## 핵심 아이디어

- `left`, `right`, `mid` 세 포인터 사용
- `arr[mid] == target`: 발견
- `arr[mid] < target`: `left = mid + 1`
- `arr[mid] > target`: `right = mid - 1`
- **전제 조건**: 배열이 정렬되어 있어야 함

| 공간 (재귀) | O(log n) |

## 동작 원리 (단계별)

```
arr = [1, 3, 5, 7, 9, 11, 13]  target = 7

Step 1: left=0, right=6, mid=3 → arr[3]=7 == 7 → 발견!

arr = [1, 3, 5, 7, 9, 11, 13]  target = 6

Step 1: left=0, right=6, mid=3 → arr[3]=7 > 6 → right=2
Step 2: left=0, right=2, mid=1 → arr[1]=3 < 6 → left=2
Step 3: left=2, right=2, mid=2 → arr[2]=5 < 6 → left=3
left > right → 없음
```

## 예시 코드

### Java
```java
int binarySearch(int[] arr, int target) {
    int left = 0, right = arr.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 오버플로 방지
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// Lower Bound: target 이상인 첫 위치
int lowerBound(int[] arr, int target) {
    int left = 0, right = arr.length;
    while (left < right) {
        int mid = (left + right) / 2;
        if (arr[mid] < target) left = mid + 1;
        else right = mid;
    }
    return left;
}
```

### JavaScript
```javascript
function binarySearch(arr, target) {
    let left = 0, right = arr.length - 1;
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

// 매개변수 탐색: 조건을 만족하는 최솟값 찾기
function parametricSearch(condition, lo, hi) {
    while (lo < hi) {
        const mid = Math.floor((lo + hi) / 2);
        if (condition(mid)) hi = mid;
        else lo = mid + 1;
    }
    return lo;
}
```

## 언제 사용하나

- 정렬된 배열에서 특정 값 탐색
- **매개변수 탐색** (Parametric Search): 최솟값/최댓값 결정
- Lower/Upper Bound 문제

---

## 주의사항

- 반드시 **정렬** 후 사용
- `mid = (left + right) / 2` → 오버플로 가능 → `left + (right - left) / 2` 권장
- while 조건: `left <= right` (exact) vs `left < right` (bound 계열) 구분
