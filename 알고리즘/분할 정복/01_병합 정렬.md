# 병합 정렬

상위개념: [[00_분할 정복]] | 인덱스: [[알고리즘 개요]]

---

## 개요

배열을 절반으로 재귀적으로 분할한 뒤, 정렬하면서 **병합(Merge)** 하는 안정 정렬 알고리즘.

---

## 핵심 아이디어

1. 배열을 반으로 나눔 (Divide)
2. 각 절반을 재귀적으로 정렬 (Conquer)
3. 두 정렬된 배열을 하나로 병합 (Combine)

---

## 자료구조

- [[배열과 리스트]] — 분할/병합 대상
- 임시 배열 (병합 시 사용)

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 최선 | O(n log n) |
| 평균 | O(n log n) |
| 최악 | O(n log n) — 항상 일정 |
| 공간 | O(n) — 임시 배열 |

## 동작 원리 (단계별)

```
[38, 27, 43, 3, 9, 82, 10]

분할: [38,27,43] | [3,9,82,10]
분할: [38] [27,43] | [3,9] [82,10]
분할: [38] [27][43] | [3][9] [82][10]

병합: [27,38,43] | [3,9,10,82]
병합: [3,9,10,27,38,43,82]
```

## 예시 코드

### Java
```java
void mergeSort(int[] arr, int left, int right) {
    if (left >= right) return;
    int mid = (left + right) / 2;
    mergeSort(arr, left, mid);
    mergeSort(arr, mid + 1, right);
    merge(arr, left, mid, right);
}

void merge(int[] arr, int left, int mid, int right) {
    int[] temp = new int[right - left + 1];
    int i = left, j = mid + 1, k = 0;
    while (i <= mid && j <= right) {
        temp[k++] = arr[i] <= arr[j] ? arr[i++] : arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= right) temp[k++] = arr[j++];
    for (int l = 0; l < temp.length; l++) arr[left + l] = temp[l];
}
```

### JavaScript
```javascript
function mergeSort(arr) {
    if (arr.length <= 1) return arr;
    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));
    return merge(left, right);
}

function merge(left, right) {
    const result = [];
    let i = 0, j = 0;
    while (i < left.length && j < right.length) {
        result.push(left[i] <= right[j] ? left[i++] : right[j++]);
    }
    return [...result, ...left.slice(i), ...right.slice(j)];
}
```

## 언제 사용하나

- **안정 정렬**이 필요할 때 (같은 값의 순서 보존)
- 최악 O(n log n)이 보장되어야 할 때
- 연결 리스트 정렬 (퀵 정렬보다 유리)
- 대용량 외부 정렬 (External Sort)

---

## 주의사항

- 추가 메모리 O(n) 필요 → 메모리 제약이 있으면 [[02_퀵 정렬]] 고려
- 재귀 깊이 O(log n) → 스택 오버플로 위험 낮음
