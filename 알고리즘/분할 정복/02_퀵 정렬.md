# 퀵 정렬

상위개념: [[00_분할 정복]] | 인덱스: [[알고리즘 개요]]

---

## 개요

**피봇(Pivot)** 을 선택하여 피봇보다 작은 원소는 왼쪽, 큰 원소는 오른쪽으로 분할한 뒤 재귀적으로 정렬하는 알고리즘. 실무에서 가장 많이 사용되는 정렬.

---

## 핵심 아이디어

1. 피봇 선택 (보통 마지막 원소, 중간값, 랜덤)
2. 파티셔닝: 피봇 기준으로 작은 것 / 큰 것 분리
3. 피봇 위치 확정
4. 양쪽을 재귀적으로 정렬

## 시간복잡도 / 공간복잡도

| | 복잡도 | 조건 |
|--|--------|------|
| 최선 | O(n log n) | 피봇이 중간값 |
| 평균 | O(n log n) | 무작위 피봇 |
| 최악 | O(n²) | 이미 정렬된 배열 + 고정 피봇 |
| 공간 | O(log n) | 재귀 스택 |

> 자세한 내용: [[시간복잡도와 공간복잡도]]

## 동작 원리 (단계별)

```
[3, 6, 8, 10, 1, 2, 1]  pivot=1(마지막)

파티셔닝:
  j=4: arr[4]=1 ≤ 1 → swap → [1, 6, 8, 10, 3, 2, 1]
  최종: swap(pivot) → [1, 1, 8, 10, 3, 2, 6]
  피봇(1) 위치 = 1

재귀: [1] | [8,10,3,2,6]
```

## 예시 코드

### Java
```java
void quickSort(int[] arr, int low, int high) {
    if (low >= high) return;
    int pivot = partition(arr, low, high);
    quickSort(arr, low, pivot - 1);
    quickSort(arr, pivot + 1, high);
}

int partition(int[] arr, int low, int high) {
    int pivot = arr[high];
    int i = low - 1;
    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            int temp = arr[i]; arr[i] = arr[j]; arr[j] = temp;
        }
    }
    int temp = arr[i+1]; arr[i+1] = arr[high]; arr[high] = temp;
    return i + 1;
}
```

### JavaScript
```javascript
function quickSort(arr, low = 0, high = arr.length - 1) {
    if (low >= high) return arr;
    const pivot = partition(arr, low, high);
    quickSort(arr, low, pivot - 1);
    quickSort(arr, pivot + 1, high);
    return arr;
}

function partition(arr, low, high) {
    const pivot = arr[high];
    let i = low - 1;
    for (let j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            [arr[i], arr[j]] = [arr[j], arr[i]];
        }
    }
    [arr[i+1], arr[high]] = [arr[high], arr[i+1]];
    return i + 1;
}
```

## 언제 사용하나

- 평균 성능이 중요하고 메모리가 제한적일 때
- in-place 정렬이 필요할 때
- 대부분의 언어 내장 정렬 기반

---

## 주의사항

- 이미 정렬된 배열 + 고정 피봇 → O(n²) 최악 → **랜덤 피봇** 사용
- 불안정 정렬 → 같은 값의 순서 보장 안 됨
- 안정 정렬이 필요하면 [[01_병합 정렬]] 사용
