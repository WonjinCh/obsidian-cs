## 알고리즘 전체 목록

### 탐색
- [[01_선형 탐색\|선형 탐색]] — O(n)
- [[03_이분 탐색\|이분 탐색]] — O(log n), 정렬 전제
- [[01_너비 우선 탐색 (BFS)\|BFS]] — O(V+E), 최단거리
- [[02_깊이 우선 탐색 (DFS)\|DFS]] — O(V+E), 모든 경로

### 완전탐색
- [[02_조합·순열 탐색\|조합·순열 탐색]] — O(n!), O(2^n)
- [[03_백트래킹\|백트래킹]] — O(k^n), 가지치기

### 정렬
- [[01_병합 정렬\|병합 정렬]] — O(n log n), 안정
- [[02_퀵 정렬\|퀵 정렬]] — O(n log n) avg, 불안정
- [[01_버블·삽입·선택 정렬\|버블·삽입·선택 정렬]] — O(n²)
- [[00_위상 정렬\|위상 정렬]] — O(V+E), DAG 전용

### 최단경로
- [[03_다익스트라\|다익스트라]] — O(E log V), 양의 가중치
- [[01_플로이드 워셜\|플로이드 워셜]] — O(V³), 모든 쌍

### 최소 신장 트리 (MST)
- [[01_크루스칼\|크루스칼]] — O(E log E), 엣지 기준
- [[02_프림\|프림]] — O(E log V), 정점 기준

## 알고리즘 선택 가이드

```
데이터 크기로 역산:
  n ≤ 10       → O(n!) — 순열 완전탐색
  n ≤ 25       → O(2^n) — 백트래킹/부분집합
  n ≤ 100      → O(n³) — 플로이드 워셜
  n ≤ 1,000    → O(n²) — DP, 기본 정렬
  n ≤ 100,000  → O(n log n) — 병합/퀵 정렬, BFS/DFS
  n ≤ 1,000,000 → O(n) — 선형탐색, 그리디

문제 유형:
  최단거리?    → 다익스트라 / BFS / 플로이드
  최소 연결?   → 크루스칼 / 프림
  순서 제약?   → 위상 정렬
  부분 문제 반복? → DP
  현재 최선 반복? → 탐욕
```
