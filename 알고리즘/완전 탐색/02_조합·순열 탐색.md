# 조합·순열 탐색

상위개념: [[00_완전 탐색]] | 인덱스: [[알고리즘 개요]]

---

## 개요

n개의 원소에서 r개를 선택하는 모든 **조합(Combination)** 또는 **순열(Permutation)** 을 열거하는 탐색 기법.

---

## 핵심 아이디어

- **조합 (nCr)**: 순서 무관, 중복 없음 → `n! / (r! × (n-r)!)`
- **순열 (nPr)**: 순서 유관 → `n! / (n-r)!`
- **부분집합**: 2^n개의 모든 부분집합 열거

---

## 자료구조

- [[배열과 리스트]] — 원소 저장 및 결과 누적

---

## 시간복잡도 / 공간복잡도

| 종류 | 시간복잡도 | 공간복잡도 |
|------|-----------|-----------|
| 조합 nCr | O(2^n) ~ O(n!/r!(n-r)!) | O(r) |
| 순열 nPr | O(n!) | O(n) |
| 부분집합 | O(2^n) | O(n) |

> 자세한 내용: [[시간복잡도와 공간복잡도]]

## 동작 원리 (단계별)

### 조합 예시: [1,2,3]에서 2개 선택
```
choose(start=0, current=[]):
  → choose(1, [1]):
      → choose(2, [1,2]) ✓ → 출력 [1,2]
      → choose(3, [1,3]) ✓ → 출력 [1,3]
  → choose(2, [2]):
      → choose(3, [2,3]) ✓ → 출력 [2,3]
```

## 예시 코드

### Java — 조합
```java
List<List<Integer>> result = new ArrayList<>();

void combine(int[] arr, int start, int r, List<Integer> current) {
    if (current.size() == r) {
        result.add(new ArrayList<>(current));
        return;
    }
    for (int i = start; i < arr.length; i++) {
        current.add(arr[i]);
        combine(arr, i + 1, r, current);
        current.remove(current.size() - 1);
    }
}
```

### Java — 순열
```java
void permute(int[] arr, boolean[] used, List<Integer> current) {
    if (current.size() == arr.length) {
        result.add(new ArrayList<>(current));
        return;
    }
    for (int i = 0; i < arr.length; i++) {
        if (!used[i]) {
            used[i] = true;
            current.add(arr[i]);
            permute(arr, used, current);
            current.remove(current.size() - 1);
            used[i] = false;
        }
    }
}
```

### JavaScript — 조합
```javascript
function combinations(arr, r) {
    const result = [];
    function combine(start, current) {
        if (current.length === r) { result.push([...current]); return; }
        for (let i = start; i < arr.length; i++) {
            current.push(arr[i]);
            combine(i + 1, current);
            current.pop();
        }
    }
    combine(0, []);
    return result;
}
```

### JavaScript — 순열
```javascript
function permutations(arr) {
    const result = [];
    const used = Array(arr.length).fill(false);
    function permute(current) {
        if (current.length === arr.length) { result.push([...current]); return; }
        for (let i = 0; i < arr.length; i++) {
            if (!used[i]) {
                used[i] = true;
                current.push(arr[i]);
                permute(current);
                current.pop();
                used[i] = false;
            }
        }
    }
    permute([]);
    return result;
}
```

## 언제 사용하나

- 모든 조합/순열을 열거해야 할 때
- n이 매우 작을 때 (조합: n ≤ 20, 순열: n ≤ 10)
- 최적 선택 집합을 찾는 문제

---

## 주의사항

- n이 크면 시간 초과 필수 → [[03_백트래킹]]으로 가지치기
- 중복 원소 처리 시 정렬 후 `if i > start && arr[i] == arr[i-1]: continue`
