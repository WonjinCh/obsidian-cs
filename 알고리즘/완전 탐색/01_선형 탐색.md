# 선형 탐색

상위개념: [[00_완전 탐색]] | 인덱스: [[알고리즘 개요]]

---

## 개요

배열/리스트를 처음부터 끝까지 **순서대로 하나씩** 검사하여 원하는 값을 찾는 가장 단순한 탐색 알고리즘.

---

## 핵심 아이디어

- 특별한 전처리(정렬 등) 불필요
- 찾으면 즉시 반환, 못 찾으면 -1(또는 None) 반환
- 최악의 경우 모든 원소 확인

---

## 자료구조

- [[배열과 리스트]] — 순차 접근 구조

---

## 시간복잡도 / 공간복잡도

| | 복잡도 |
|--|--------|
| 최선 | O(1) — 첫 번째가 정답 |
| 평균 | O(n/2) = O(n) |
| 최악 | O(n) — 마지막 또는 없음 |
| 공간 | O(1) |

> 자세한 내용: [[시간복잡도와 공간복잡도]]

---

## 동작 원리 (단계별)

```
[3, 1, 4, 1, 5, 9]  target = 5

Step 1: index 0, arr[0]=3 ≠ 5
Step 2: index 1, arr[1]=1 ≠ 5
Step 3: index 2, arr[2]=4 ≠ 5
Step 4: index 3, arr[3]=1 ≠ 5
Step 5: index 4, arr[4]=5 = 5  → 반환 4
```

---

## 예시 코드

### Java
```java
public static int linearSearch(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) return i;
    }
    return -1;
}
```

### JavaScript
```javascript
function linearSearch(arr, target) {
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] === target) return i;
    }
    return -1;
}
```

## 언제 사용하나

- 데이터가 정렬되지 않았을 때
- 데이터 크기가 매우 작을 때 (n ≤ 100)
- 한 번만 탐색하는 경우 (정렬 비용이 더 클 때)
